## Usage

This is a MUI Link that can be anywhere links are needed.

## Import

```
import Link from '@mui/material/Link';
import { ThemeProvider } from '@mui/material'; <---- allows us to apply a custom theme
import { theme } from '#styles/styles'; <---- custom theme with Progress branding

```

## Variations

### Underline

Adjust the size using the `size` prop. Support is available for 3 sizes:

| underline | link                                                                                                                                        |
| --------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| always    | <ThemeProvider theme={theme}><Link style={{ cursor: 'pointer'}} underline='always' variant="body1"> Always Underline</Link></ThemeProvider> |
| hover     | <ThemeProvider theme={theme}><Link style={{ cursor: 'pointer'}} underline='hover' variant="body1"> Hover Underline</Link></ThemeProvider>   |
| none      | <ThemeProvider theme={theme}><Link style={{ cursor: 'pointer'}} underline='none' variant="body1"> No Underline </Link></ThemeProvider>      |
