## Usage

This is a MUI-based Typography component that can be used or anywhere else text is needed.

## Import

```
import Typography from '@mui/material/Typography';
import { ThemeProvider } from '@mui/material'; <---- allows us to apply a custom theme
import { theme } from '#styles/styles'; <---- custom theme with Progress branding

```
