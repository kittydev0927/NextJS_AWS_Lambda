## Usage

This component is a MUI Checkbox to be used to build forms where checkboxes are needed for selection.

## Import

```
import { Checkbox } from '#components/forms/atoms/Checkbox/Checkbox';

```

## Variations

### Size

Adjust the size using the `size` prop. Support is available for 3 sizes:

| size   | checkbox                                                                |
| ------ | ----------------------------------------------------------------------- |
| small  | <ThemeProvider theme={theme}><Checkbox size='small' /></ThemeProvider>  |
| medium | <ThemeProvider theme={theme}><Checkbox size='medium' /></ThemeProvider> |
| large  | <ThemeProvider theme={theme}><Checkbox size='large' /></ThemeProvider>  |

### MUI Checkbox Docs - https://mui.com/components/checkboxes/
