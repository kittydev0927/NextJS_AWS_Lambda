## Usage

This is a MUI Button that can be used for forms (or anywhere else buttons are needed).

## Import

```
import Button from '@mui/material/Button';
import { ThemeProvider } from '@mui/material'; <---- allows us to apply a custom theme
import { theme } from '#styles/styles'; <---- custom theme with Progress branding

```

## Variations

### Size

Adjust the size using the `size` prop. Support is available for 2 sizes:

| size  | button                                                                                                      |
| ----- | ----------------------------------------------------------------------------------------------------------- |
| small | <ThemeProvider theme={theme}><Button size='small' variant="primary"> Small Button</Button></ThemeProvider>  |
| large | <ThemeProvider theme={theme}><Button size='large' variant="primary"> Large Button </Button></ThemeProvider> |

### Variats

Adjust the colors using the `variant` prop. Support is available for 3 variants:

| Variant   | button                                                                                      |
| --------- | ------------------------------------------------------------------------------------------- |
| primary   | <ThemeProvider theme={theme}><Button variant="primary">Primary</Button></ThemeProvider>     |
| secondary | <ThemeProvider theme={theme}><Button variant="secondary">Secondary</Button></ThemeProvider> |
| inactive  | <ThemeProvider theme={theme}><Button variant="inactive">Inactive</Button></ThemeProvider>   |
