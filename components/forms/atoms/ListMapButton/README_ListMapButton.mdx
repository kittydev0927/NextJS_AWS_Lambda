## Usage

This component is list or map display view of saved homes. This components uses toggle buttons with exclusive selection.

## Import

```
import { ListMapButton } from '#components/forms/atoms/ListMapButton/ListMapButton';
```

### Disabled

Set `disabled` prop to `true` or `false`

| disabled | ListMapButton                                                                                                              |
| -------- | -------------------------------------------------------------------------------------------------------------------------- |
| true     | <LocalizationProvider dateAdapter={AdapterDateFns}><ListMapButton disabled={true}></ListMapButton></LocalizationProvider>  |
| false    | <LocalizationProvider dateAdapter={AdapterDateFns}><ListMapButton disabled={false}></ListMapButton></LocalizationProvider> |
